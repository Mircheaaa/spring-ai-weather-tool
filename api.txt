# 기상청 중기예보 조회서비스 Open API 활용 가이드 (AI 학습용)

## 1. 서비스 개요

- **API명 (국문):** 중기예보 조회서비스
- **API명 (영문):** MidFcstInfoService
- **API 설명:** 중기전망, 중기육상예보, 중기기온, 중기해상예보 정보를 조회하는 서비스입니다.
- **서비스 URL:** http://apis.data.go.kr/1360000/MidFcstInfoService
- **인증 방식:** 서비스키(ServiceKey) 사용
- **인터페이스:** REST (GET)
- **교환 데이터 표준:** XML, JSON
- **데이터 갱신 주기:** 매일 2회 (06시, 18시)

---

## 2. 상세 기능 목록 (API 목록)

1.  **getMidFcst (중기전망조회):** 기상 전망 정보를 조회합니다.
2.  **getMidLandFcst (중기육상예보조회):** 육상 날씨 예보(강수 확률, 날씨)를 조회합니다.
3.  **getMidTa (중기기온조회):** 최저/최고 기온 예보를 조회합니다.
4.  **getMidSeaFcst (중기해상예보조회):** 해상 날씨 및 파고 정보를 조회합니다.

---

## 3. 기능별 명세

### 3.1. 중기전망조회 (getMidFcst)

- **기능 설명:** 지점 번호와 발표 시각을 기준으로 기상 전망 정보를 조회합니다.
- **요청 URL:** http://apis.data.go.kr/1360000/MidFcstInfoService/getMidFcst

#### 요청 파라미터 (Request)
- **serviceKey (인증키):** [필수] 공공데이터포털에서 발급받은 인증키 (URL 인코딩 필요)
- **numOfRows (한 페이지 결과 수):** [옵션] 한 페이지에 보여줄 결과 수 (기본값: 10)
- **pageNo (페이지 번호):** [옵션] 조회할 페이지 번호 (기본값: 1)
- **dataType (응답자료형식):** [옵션] 응답 데이터 형식 (XML/JSON, 기본값: XML)
- **stnId (지점번호):** [필수] 조회할 지점 번호 (하단 '지점코드' 참조)
- **tmFc (발표시각):** [필수] 'YYYYMMDDHHMM' 형식의 발표 시각 (예: 202401010600). 06시, 18시에 생성되며 최근 24시간 자료만 제공.

#### 응답 파라미터 (Response)
- **wfSv (기상전망):** 텍스트 형식의 기상 전망 정보.

---

### 3.2. 중기육상예보조회 (getMidLandFcst)

- **기능 설명:** 예보 구역 코드와 발표 시각을 기준으로 4~10일 후의 육상 날씨 정보를 조회합니다.
- **요청 URL:** http://apis.data.go.kr/1360000/MidFcstInfoService/getMidLandFcst

#### 요청 파라미터 (Request)
- **serviceKey, numOfRows, pageNo, dataType:** 위와 동일
- **regId (예보구역코드):** [필수] 조회할 예보 구역 코드 (하단 '중기육상예보구역 코드' 참조)
- **tmFc (발표시각):** [필수] 위와 동일

#### 응답 파라미터 (Response)
- **rnSt[4-10][Am/Pm] (강수 확률):** 4~10일 후의 오전/오후 강수 확률 (%). 8일 후부터는 오전/오후 구분 없음.
- **wf[4-10][Am/Pm] (날씨 예보):** 4~10일 후의 오전/오후 날씨 예보 (맑음, 구름많음 등). 8일 후부터는 오전/오후 구분 없음.
- **참고:** 06시 발표는 4~10일, 18시 발표는 5~10일 예보를 제공합니다.

---

### 3.3. 중기기온조회 (getMidTa)

- **기능 설명:** 예보 구역 코드와 발표 시각을 기준으로 4~10일 후의 최저/최고 기온 정보를 조회합니다.
- **요청 URL:** http://apis.data.go.kr/1360000/MidFcstInfoService/getMidTa

#### 요청 파라미터 (Request)
- **serviceKey, numOfRows, pageNo, dataType:** 위와 동일
- **regId (예보구역코드):** [필수] 조회할 예보 구역 코드 (하단 '중기기온예보구역 코드' 참조)
- **tmFc (발표시각):** [필수] 위와 동일

#### 응답 파라미터 (Response)
- **taMin[4-10] (예상 최저기온):** 4~10일 후의 예상 최저기온(℃).
- **taMax[4-10] (예상 최고기온):** 4~10일 후의 예상 최고기온(℃).
- **taMin[4-10]Low/High (최저기온 범위):** 예상 최저기온의 하한/상한 범위.
- **taMax[4-10]Low/High (최고기온 범위):** 예상 최고기온의 하한/상한 범위.
- **참고:** 06시 발표는 4~10일, 18시 발표는 5~10일 예보를 제공합니다.

---

## 5. 에러 코드

- **00:** NORMAL_SERVICE (정상)
- **01:** APPLICATION_ERROR (어플리케이션 에러)
- **02:** DB_ERROR (데이터베이스 에러)
- **03:** NODATA_ERROR (데이터없음 에러)
- **04:** HTTP_ERROR (HTTP 에러)
- **05:** SERVICETIME_OUT (서비스 연결실패 에러)
- **10:** INVALID_REQUEST_PARAMETER_ERROR (잘못된 요청 파라미터 에러)
- **11:** NO_MANDATORY_REQUEST_PARAMETERS_ERROR (필수요청 파라미터 없음)
- **12:** NO_OPENAPI_SERVICE_ERROR (서비스가 없거나 폐기됨)
- **20:** SERVICE_ACCESS_DENIED_ERROR (서비스 접근거부)
- **22:** LIMITED_NUMBER_OF_SERVICE_REQUESTS_EXCEEDS_ERROR (서비스 요청제한횟수 초과)
- **30:** SERVICE_KEY_IS_NOT_REGISTERED_ERROR (등록되지 않은 서비스키)
- **31:** DEADLINE_HAS_EXPIRED_ERROR (기한만료된 서비스키)
- **32:** UNREGISTERED_IP_ERROR (등록되지 않은 IP)
- **99:** UNKNOWN_ERROR (기타 에러)
